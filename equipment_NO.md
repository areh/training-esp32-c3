# Utstyr


## Basis

* Mikrokontroller, ESP32-C3
* USB-C kabel
* Breadboard
* Dupont kabler
* Motstander,  220 Ohm og 4,7 kOhm


## Output

* Lysdioder (4x)
* Nanopixel (8x LED) - WS2812B
* 7-segment display (8 siffer) - MAX7219


## Input

* Temperatursensor - DS18B20
* Bevegelsessensor - PIR sensor
* Knapp
* Potentiometer
