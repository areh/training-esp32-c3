# Equipment


## Basics

* Microcontroller, ESP32-C3
* USB-C cable
* Breadboard
* Dupont cables


## Output

* LEDs (Light Emitting Diodes)
* Neopixel - WS2812B
* 7-segment display (8 digits) - MAX7219


## Input

* Temperature sensor - DS18B20
* Movement sensor - PIR sensor
* Button
* Potentiometer
